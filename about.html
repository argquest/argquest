Köpenick Quest

This is the alternative reality game that takes place in the Köpenick. The game is linked to different tourist attractions. Multiple locations are connected through various puzzles. To get a puzzle you have to scan the QR-code that you find in the previous location.

The game is still in development. This page will be updated once game is finished and link to the first puzzle will be posted here.
